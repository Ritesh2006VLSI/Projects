# 🚀 Verilog HDL Projects

A collection of Digital Design and RTL projects implemented using **Verilog HDL**. This repository showcases my journey in learning and applying concepts of **Digital Logic Design**, **Finite State Machines (FSMs)**, and **RTL Development**.

## 🎯 Objectives

* Strengthen Verilog HDL and RTL design skills
* Implement industry-relevant digital systems
* Practice modular and hierarchical design techniques
* Develop verification skills using testbenches and simulation
* Build a strong portfolio for VLSI and semiconductor roles

---

## 📂 Projects

### 🔹 Universal Shift Register

* 4-bit Universal Shift Register
* Hold, Parallel Load, Shift Left, Shift Right operations
* Synchronous design
* Testbench verification

### 🔹 Digital Clock

* HH:MM:SS digital clock
* Clock divider for 1 Hz generation
* Modular counter-based design
* Simulation verified

### 🔹 Traffic Light Controller (FSM)

* FSM-based traffic signal controller
* Pedestrian crossing support
* Emergency vehicle priority
* Minimum green-time enforcement

### 🔹 4-bit ALU using 8:1 Multiplexer

* Arithmetic and logical operations
* Modular Verilog implementation
* Functional verification through simulation

### 🔹 4-bit Comparator using Multiplexer

* Hierarchical comparator design
* Bitwise comparison from MSB to LSB
* Testbench and waveform verification
